Chapter 1: Introduction

This is a sample HTML book used for testing the ingestion pipeline. It contains simple paragraphs and headings to verify that the HTML parser correctly extracts text content.

Background

The interactive books project supports multiple file formats. HTML is one of the supported formats, parsed as a single logical page.

Chapter 2: Details
